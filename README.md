# DChat (decentralized chat) - contracts

Smart contracts written in solidity with the help of [hardhat](https://hardhat.org/getting-started/)

### How to use

#### Install tool and modules

`npm i`

#### Test contracts (mocha/chai)

`npm run test`

#### Compile

`npm run compile`

### Hardhat tasks

### Generate ABI in json

`hh gen-abi dir`
